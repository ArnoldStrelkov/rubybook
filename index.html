Изучаем ROR и JS на интересном проекте

Мы будем делать сервис по поиску единомышленников. Это такой сайт на котором каждый пользователь может разместить свою идею, предложение, проект, а другие пользователи сайта могут к данному проекту присоедениться.

Проверка залогинившегося юзера

Перед выполнением любого метода в контроллерах с помощью вызова before_action :check_login мы вызываем метод check_login

class ApplicationController < ActionController::Base
  # Prevent CSRF attacks by raising an exception.
  # For APIs, you may want to use :null_session instead.
  protect_from_forgery with: :exception
  before_action :check_login

  private

  def check_login

    @current_user ||= User.where(id: session[:user_id]).first

  end  

end

Метод check_login проверяет есть ли в куках запись сеcсии с номером id пользователя и если есть то находит этого юзера и записывает его в переменную @current_user. Данная переменная будет доступна в наших контроллерах и вьюхах.

Вывод списка идей

По умолчанию у нас стоит роут root 'main#index' котрый передает управление методу index в контроллере main

class MainController < ApplicationController
  def index
    @ideas = Idea.all.order(id: :desc)
  end
end

Данный метод просто запрашивает все записи из модели Idea в переменную @ideas. Затем управление передается во выюху index.html.erb

Заводим модель User

Модель юзера заводим с помошью такой миграции

class CreateUsers < ActiveRecord::Migration
  def change
    create_table :users do |t|
      t.string :name
      t.string :avatar
      t.string :provider
      t.string :uid
      t.string :token
    end
  end
end

В поля provider и uid мы будем записывать информацию полученную от социальной сети при авторизации. Ну и затем подрубаем авторизацию. Для этого прописываем в файле Gemfile два гема 

gem 'omniauth'
gem 'omniauth-vkontakte'

прогоняем команду bundle install в папке config\initializers создаем файл omniauth.rb и прописываем в него следующие строчки

Rails.application.config.middleware.use OmniAuth::Builder do
    provider :vkontakte, ENV['VK_KEY'], ENV['VK_SECRET']
  end
