Заводим модель User

Модель юзера заводим с помошью такой миграции

class CreateUsers < ActiveRecord::Migration
  def change
    create_table :users do |t|
      t.string :name
      t.string :avatar
      t.string :provider
      t.string :uid
      t.string :token
    end
  end
end

В поля provider и uid мы будем записывать информацию полученную от социальной сети при авторизации. Ну и затем подрубаем авторизацию. Для этого прописываем в файле Gemfile два гема 

gem 'omniauth'
gem 'omniauth-vkontakte'

прогоняем команду bundle install в папке config\initializers создаем файл omniauth.rb и прописываем в него следующие строчки

Rails.application.config.middleware.use OmniAuth::Builder do
    provider :vkontakte, ENV['VK_KEY'], ENV['VK_SECRET']
  end
